# GuessMyMBTI
### Front End -서아름 -임수빈
### Back End -양종욱 -남윤형

### 개요
MBTI로 소통가능한 웹서비스 페이지


## Getting Started

```
www.gmmclients.click
```

## Deployment

AWS EC2, S3, CodeDeploy & GitHub Actions

## Built With

* Spring Boot Gradle
* React

## Back End - Spring Boot 사용 기술
- Spring Security
- Kakao Social Login
- JWT Authentication
- REST API 설계 기반
- AWS 배포 및 GitHub Actions 와 CD 구축
